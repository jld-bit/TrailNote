# TrailNote (SwiftUI + SwiftData + MapKit)

TrailNote is an original iPhone app concept for planning trips and organizing stops on a map.

## Features
- SwiftUI + MVVM structure
- SwiftData local persistence
- Onboarding with sample trip seed
- Trip creation (title, icon cover, dates, notes)
- Trip stops with manual reorder and map pins
- Trip checklist for packing/tasks
- Sharing summary as text or image
- StoreKit 2 premium gating:
  - Free tier: limited trips
  - Premium: unlimited trips, custom colors, export image, offline packs messaging
